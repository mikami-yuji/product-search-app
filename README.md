# 商品検索ツール

Excelファイルから商品データを読み込み、検索・フィルタリング・カート機能を備えたAmazonスタイルのWebアプリケーションです。

## 主な機能

### 📊 データ管理
- **Excelファイル読み込み** - .xlsx/.xlsファイルをドラッグ&ドロップまたは選択
- **データキャッシュ** - IndexedDBを使用した自動保存
- **画像フォルダ接続** - File System Access APIでローカル画像を表示

### 🔍 検索・フィルター
- **キーワード検索** - 全項目を横断検索
- **プルダウンフィルター** - 種別、重量、材質、総色数、直送先で絞り込み
- **ソート機能** - 価格順（安い/高い）、最新受注日順

### 🎨 表示機能
- **グリッドビュー** - Amazonスタイルの商品カード表示
- **テーブルビュー** - 詳細な一覧表示
- **画像ギャラリー** - 複数画像（A、B、C...）の切り替え表示
- **Web画像対応** - asahipac.co.jpからの画像取得
- **経過期間表示** - 22ヶ月以上は黄色、24ヶ月以上は赤色でハイライト

### 🛒 カート機能
- **カートに追加** - 前回注文数を初期値として設定
- **数量調整** - 100単位での増減
- **メール文章生成** - カート内容をメール用テキストに変換してコピー

### 📱 UI/UX
- **レスポンシブデザイン** - PC・タブレット・スマホ対応
- **ページネーション** - 20件ずつ表示
- **詳細モーダル** - 商品情報をセクション分けして整理表示
- **アニメーション** - スムーズなトランジション効果

## 技術スタック

- **React** - UIライブラリ
- **Vite** - ビルドツール
- **XLSX** - Excelファイル解析
- **idb-keyval** - IndexedDB操作
- **Lucide React** - アイコン
- **Cheerio & Axios** - Web画像スクレイピング（Node.js）

## セットアップ

### 必要要件
- Node.js 16以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd product-search-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

### Web画像取得（オプション）

```bash
# Web画像のURLを取得（asahipac.co.jp）
node fetch_images.js

# public/product_images.jsonが生成されます
```

## 使い方

1. **Excelファイルを読み込む**
   - 右上の「ファイル選択」ボタンからExcelファイルを選択
   - データが自動的にキャッシュされます

2. **画像フォルダを接続**（オプション）
   - 「画像フォルダ」ボタンをクリック
   - ローカルの画像フォルダを選択
   - 受注№に対応する画像（61858.jpg、61858A.jpg等）が表示されます

3. **商品を検索・フィルター**
   - 上部の検索バーでキーワード検索
   - 左サイドバーでフィルター条件を選択
   - ソートドロップダウンで並び替え

4. **表示モードを切り替え**
   - グリッドビュー: カード形式で表示
   - テーブルビュー: 一覧形式で表示

5. **商品詳細を確認**
   - 商品カードまたは行をクリック
   - モーダルで詳細情報と画像ギャラリーを表示
   - 左右ボタンで複数画像を切り替え

6. **カートに追加して注文**
   - 「カートに追加」ボタンをクリック
   - ヘッダーの「カート」ボタンでカートを表示
   - 数量を調整（+/-ボタンは100単位）
   - 「メール文章をコピー」で注文内容をコピー
   - メールソフトに貼り付けて送信

## Excelファイルの形式

以下の列を含むExcelファイルに対応しています:

- 受注№
- 種別
- タイトル
- 商品コード
- 形状
- 重量
- 材質名称
- 受注数
- 単価
- 印刷代
- 表色数
- 裏色数
- 総色数
- JANコード
- 直送先名称
- 最新受注日

## ブラウザ対応

- Chrome/Edge 86+（推奨）
- Firefox 90+
- Safari 15.4+

※ File System Access APIを使用するため、最新のモダンブラウザが必要です
